TRLRequest

@interface TRLRequest : NSObject

TRLRequest is the end point for our Request

Only thing that can be mutated here is the Parameters they can be added and removed based on the Trolley request that is been created.

Please use like follows inside the internal code:

@code TRLURLDataRequest *req = [trlRequest request]; [req response:^(NSData *data, NSError *error) { … }]; @endcode

Warning

Please don’t write to HTTPMethod, ran into an initalisation error so is not readonly
